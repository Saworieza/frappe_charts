This gem includes the [Frapp√© Charts](https://github.com/frappe/charts) libraries in your Rails application.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'frappe_charts'
```

And then execute:

    $ bundle

## Usage

Add `//= require frappe-charts.min.iife` to your application.js file and
`*= require frappe-charts.min` to your application.css file.
